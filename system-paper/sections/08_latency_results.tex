\subsection{The FP16-GPU Context Wall}
% Deterministic decode failure (asterisk repetition loop) at ~5,000
% total context tokens on FP16 GPU attention; root-caused to precision
% asymmetry between GPU and CPU paths, not a model-artifact defect.
% FP32-GPU fallback verified clean past 5,000 tokens at ~25% TTFT cost.
% Deployment ceiling set to 4,096 tokens (~900-token safety margin).
% Give this finding space -- it is novel and load-bearing for anyone
% deploying LiteRT-LM-class runtimes at longer contexts.
